Keep Talking and Nobody Explodes Mod Face Perception

On the Subject of Face Perception

I hope you have no prosopagnosia.

The module consists of a "NEXT" button and a screen that displays a face and a number.

Remember this face and number. Then press "NEXT" button. You will see a different face and a different number. Continue to press "NEXT", remembering the faces and numbers, until the screen changes to a list of names. Using the tables below, match the faces you see with the names. Count the sum of all remembered numbers for each name. Press five names in decreasing order of the sum. If several names have the same sum, then the name that got this sum first should be pressed first. In case of an incorrect answer, you will receive a strike and the module will be reset.
